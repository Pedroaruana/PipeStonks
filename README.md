# Wasteland Garden

Gerenciador de tarefas gamificado em pixel art. A história se passa em 2056, num mundo pós-apocalíptico onde o ar é irrespirável. Você cultiva plantas para sobreviver — cada tarefa é uma semente que cresce até virar uma árvore e gerar frutos.

## Stack

- Frontend: React + Vite
- Backend: Node.js + Fastify
- Banco: PostgreSQL (Supabase)
- Testes: Vitest, Playwright
